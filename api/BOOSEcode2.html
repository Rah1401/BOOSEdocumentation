Table of Contents

Namespace BOOSEcode2

Classes

BOOSEform

Main Windows Form for running BOOSE programs.

MyArray

Custom array implementation that removes BOOSE size restrictions. Supports both integer and real (double) array types in 1D or 2D configurations. Works with peek and poke operations for array element access.

MyBoolean

Custom Boolean variable class that removes BOOSE restrictions. Handles boolean values and logical expressions.

MyCall

Custom call command for invoking defined methods. Removes BOOSE restrictions on method calls and argument passing. Syntax: call methodName [arg1, arg2, ...]

MyCanvas

A simple canvas used by the BOOSE interpreter. Implements Singleton pattern to ensure only one canvas instance exists. Handles drawing shapes, lines, colours, text, and cursor movement.

MyCircle

Replacement circle command that uses our canvas and does not have the BOOSE radius restriction.

MyClear

Custom clear command that clears the canvas.

MyCommandFactory

Custom command factory that creates custom BOOSE commands. Replaces specific BOOSE commands with custom implementations while allowing other standard BOOSE commands to work normally.

MyElse

Custom else command for if-else branching. Marks the transition from if-true block to if-false block. When the if condition is true, execution jumps over the else block.

MyEnd

Custom end command that closes if, while, for, and method blocks. Removes BOOSE nesting and complexity restrictions. Handles loop continuation and method returns.

MyEvaluation

Custom evaluation class for handling standalone variable assignments. Allows assignments like "width = 2*radius" after variable declaration.

MyFor

Custom for loop implementation that removes BOOSE iteration restrictions. Supports: for variable = start to end [step increment] Default step is 1 if not specified.

MyIf

Custom If command that removes nesting restrictions. Evaluates a boolean condition and executes commands conditionally.

MyInt

Custom integer variable implementation that removes BOOSE restrictions. Supports unlimited variable names and no count limits. Handles both declaration and initialization syntax.

MyMethod

Custom method command for defining reusable procedures. Removes BOOSE restrictions on method complexity and parameter count. Syntax: method returnType methodName [paramType paramName, ...]

MyMoveTo

Custom moveto command that simply forwards to our canvas.

MyParser

Custom parser that extends BOOSE's Parser to handle custom variable types. Intercepts int and real declarations to use custom classes, and handles standalone variable assignments.

MyPeek

Custom peek command for reading values from array elements. Inherits from MyArray to access array operations. Syntax: peek variable = arrayName row col (2D) or peek variable = arrayName row (1D)

MyPoke

Custom poke command for writing values into array elements. Inherits from MyArray to access array operations. Syntax: poke arrayName row col = value (2D) or poke arrayName row = value (1D)

MyReal

Custom real number variable implementation that removes BOOSE restrictions. Supports floating-point arithmetic and unlimited variable declarations. Handles variable substitution and expression evaluation.

MyRect

Custom rectangle command that draws a rectangle using the application's canvas instead of BOOSE's default one.

MyTri

Replacement triangle command that uses our canvas and does not have the BOOSE restrictions.

MyWhile

Custom While loop implementation that removes BOOSE restrictions. Executes a block of commands repeatedly while a condition remains true. Syntax: while [condition] ... endwhile

MyWrite

Custom write command for displaying text and variable values on the canvas. Supports variable substitution and expression evaluation for int, real, and boolean types.
